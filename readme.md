# Contributing / Reading Locally

### "I've never used Github in my life, what do I do?"
TL;DR,

IF YOU ARE PLANNING TO CONTRIBUTE:
- Create an account on Github.
- Grab a Git client of your choosing. If you run Windows, I'd suggest [Github Desktop](https://desktop.github.com/download/) if you just want to do this with no major, or [Fork](https://git-fork.com/) for more control if you want to get into the weeds of it.
- Click the 'fork' button on [this repository](https://github.com/unit0016/MourningDew-Conspiracy-Board) and clone your fork locally. (Top left of Github Desktop's UI, Add, Clone.)
- Create a new branch, and title it whatever you like.
- Download Obsidian [here](https://obsidian.md/download).
- Open the repository in Obsidian.
- Make whatever changes you wish.
- In your Git Client, hit commit.
- In 'Branch', you should have the option to open a pull request from that branch.
- Describe your changes, screenshot any relevant sections; etc et al - I'll review them and get to them when I can.

IF YOU JUST WANT TO BROWSE THIS LOCALLY:
- I would still suggest you use a Git client; and [clone this repository](https://github.com/unit0016/MourningDew-Conspiracy-Board) without forking it.
  - *You can, however, alternatively simply download this repository and view it in obsidian; though you won't be able to keep it up to date as easily.*
- You will still need [Obsidian](https://obsidian.md/download) in order to view the Vault itself. 

### Make sure to keep your 'main' branch up to date!
- If you forked the repository; Branch -> Pull from upstream/main.
- If you didn't fork the repository, simply fetching / pulling will bring down the current changes.
